Product concept and feature selection

Define what the product should look like and its functions.

Outline the value it offers. To verify this, create an MVP (Minimum Viable Product) to test your concept and list the necessary features to add to the roadmap. Features and functions can include:

  • Key features

  • Additional features

  • Visual features (interface, branding, integrations)
